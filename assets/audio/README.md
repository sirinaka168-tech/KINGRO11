# เพลงพื้นหลัง (BGM)

`bgm.mp3` คือเพลงที่เล่นในเว็บ (วนลูป) — กดปุ่มโน้ตเพลงมุมขวาล่างเพื่อเปิด/ปิด

**เพลงปัจจุบัน:** "Folk Round" by Kevin MacLeod (incompetech.com)
ลิขสิทธิ์ [CC BY 3.0](https://creativecommons.org/licenses/by/3.0/) — ใช้ได้ฟรีถ้าให้เครดิต
(เครดิตอยู่ใน footer ของเว็บแล้ว)

## อยากเปลี่ยนเพลงเอง?

วางไฟล์ทับชื่อ `bgm.mp3` ได้เลย เช่นเพลง RO OST ("Theme of Prontera" ฯลฯ)

> ⚠️ ถ้าใช้เพลง RO จริง = ลิขสิทธิ์ Gravity/SoundTeMP — ใช้เฉพาะเซิร์ฟส่วนตัว/ไม่พาณิชย์
> ถ้าจะเปิดสาธารณะ แนะนำใช้เพลง royalty-free (incompetech, Pixabay Music, YouTube Audio Library)

หมายเหตุ: เบราว์เซอร์บล็อกการเล่นเพลงอัตโนมัติ — เพลงจะเริ่มเล่นหลังผู้ใช้
คลิก/แตะหน้าจอครั้งแรก หรือกดปุ่มเพลงเอง
